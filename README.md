# Playwright E-commerce Automation
This project demonstrates end-to-end automation testing using Playwright with the Page Object Model.

## Tech Stack
- Playwright
- JavaScript
- Page Object Model
- GitHub Actions

## Project Structure
pages/
Contains page object classes.

pageObject/
Contains PoManager to manage page objects.

tests/
Contains Playwright test cases.

## Test Scenario
Search product on Flipkart.

## How to Run Tests
Install dependencies
npm install
Run tests
npx playwright test
